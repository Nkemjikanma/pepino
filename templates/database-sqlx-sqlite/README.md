### Prerequisites
    - Rust - Node.js - PostgreSQL - just - sqlx-cli - typeshare-cli

### Quick start
   ##### Clone and setup
   cd my-app-name
   cp .env.example .env

   ###### Install dependencies
   just install

   ###### Run (in separate terminals)
   just dev-server
   just dev-client

   ###### Generate types
   just generate-types
